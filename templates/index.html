NN_EOS

This is a web application for training neural network surrogates for thermodynamic property prediction. Upload the dataset, configure the training parameters, and download a trained model ready for integration into CFD solvers.

Traditional equation of state (EOS) libraries are accurate but slow. They use very complex functions to predict thermodynamic properties like phasic densities, speed of sound etc. Evaluation of these functions are computationally expensive and slows down the simulation. A neural network surrogate trained on EOS data can achieve comparable accuracy at significantly higher speed — making it suitable for integration into time-critical CFD simulations.

Workflow

  • Accepts a CSV file with thermodynamic data (P, T, property columns)
  • Scales the data and trains a neural network surrogate
  • Shows training results — MAPE, loss curve, parity plot
  • Returns a downloadable TensorFlow SavedModel
  • Can resume the training from last saved model also
