nn_eos — Neural Network Surrogate Trainer

nn_eos is a web application for training neural network surrogates for thermodynamic property prediction. Upload your dataset, configure the training parameters, and download a trained model ready for integration into CFD solvers.

What it does

  • Accepts a CSV file with thermodynamic data (P, T, property columns)
  • Scales the data and trains a neural network surrogate
  • Shows training results — MAPE, loss curve, parity plot
  • Returns a downloadable TensorFlow SavedModel

Use case

Traditional equation of state (EOS) libraries are accurate but slow. A neural network surrogate trained on EOS data achieves comparable accuracy at significantly higher speed — making it suitable for integration into time-critical CFD simulations.
